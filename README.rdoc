== UnTube

Dead simple video service.

=== Service

Ruby: <tt>~> 2.2</tt>
Rails: <tt>~> 4.2</tt>
Rspec: <tt>~> 3.4</tt>

=== System::

PostgreSQL:: <tt>~> 9.5<tt>

=== Setup

Database creation:: <tt>bundle exec db:setup</tt>

Database initialization:: <tt>bundle exec db:migrate</tt>

How to run the test suite:: <tt>bundle exec rake spec</tt>
